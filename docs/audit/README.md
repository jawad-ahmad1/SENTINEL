# Audit Reports

This directory contains the professional audit and remediation outputs for the Sentinel codebase.

## Reports

- `executive-summary.md` - management summary, score, and production-readiness decision.
- `full-audit-report.md` - complete technical findings and subsystem analysis.
- `remediation-log.md` - prioritized issues and concrete fixes.
- `validation-test-plan.md` - integration/automation/manual validation plan.
- `metrics-dashboard.md` - codebase metrics and scoring dashboard.
- `technical-debt-register.md` - debt backlog with effort/ROI.

## Artifacts

Machine-generated analysis data is in `artifacts/`:

- `file-manifest.csv`
- `file-inventory.csv`
- `complexity-analysis.csv`
- `duplication-analysis.csv`
- `metrics.json`

## Naming Convention

- Use lowercase kebab-case for new report files.
- Keep ad-hoc or temporary local outputs out of the repository root.
