# Screenshots

This directory contains screenshots of the Sentinel RFID Attendance System for documentation purposes.

## Required Screenshots

Capture and add the following screenshots:

| Filename | Description | Page |
|----------|-------------|------|
| `kiosk_idle.png` | Kiosk waiting for RFID scan | `index.html` |
| `kiosk_checkin.png` | Kiosk after successful check-in | `index.html` |
| `kiosk_checkout.png` | Kiosk after check-out | `index.html` |
| `login.png` | Admin login page | `login.html` |
| `dashboard.png` | Admin dashboard with live stats | `admin.html` |
| `employees.png` | Employee management table | `employees.html` |
| `register.png` | Employee registration form | `register.html` |
| `reports_daily.png` | Daily attendance report | `reports.html` |
| `reports_monthly.png` | Monthly attendance report | `reports.html` |
| `settings.png` | System settings page | `settings.html` |

## How to Capture

1. Open the page in Chrome
2. Press F12 → toggle device toolbar for consistent viewport
3. Set viewport to 1920×1080
4. Take screenshot (Ctrl+Shift+P → "Capture full size screenshot")
5. Save to this directory with the filename from the table above
